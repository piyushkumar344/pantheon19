CODEZILLA


Event Description

It is a team programming contest which will be conducted in two stages. In the first stage, an online contest will be conducted on HackerEarth where top 30 teams from BIT and top 20 from other colleges will be selected. Although there would be a common rank list for all years, top 5 teams from first year will be selected to go to the next round as well. There would be multiple problems and points would be allotted on solving the problems totally.

Day:   1

Time:   11:00 AM

Venue:  LH1

Points:  250-200-150

Rules and Regulations

  • The contest will have the ACM ICPC Style Rank List. th
  • There is no restriction on the programming language.
  • All the problems have same points allotted to them.
  • Users are ranked according to the most problems solved. Ties will be broken by the total time for each user in ascending order of time.
  • The total time is the sum of the time consumed for each problem solved. The time consumed for a solved problem is the time elapsed from the beginning of the contest to the submission of the first accepted run plus 20 penalty minutes for every previously rejected run for that problem (Wrong Answer, Time Limit Exceeded, Runtime Error, etc).
  • There is no time consumed for a problem that is not solved.
  • The decision of the organizers in declaring the results will be final. No queries in this regard will be entertained.
  • Any participant found to be indulging in any form of malpractice will be immediately disqualified.
